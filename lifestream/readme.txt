== Description ==

Lifestream displays your social feeds and photos much like you would see it on many of the social networking sites.

== Installation ==

Drop lifestream folder into /wp-content/plugins/ and activate the plug in.

== Usage ==

Just put the following code in a WordPress page:

`<?php Lifestream(); ?>`

== Requirements ==

* PHP5 (hey, PHP4 is doomed!)
* (Recommended) wp-exec plugin

== Features ==

* Uses WordPress resources (no extra libraries needed)
* Personalizable CSS classes
* Options Page
* Detects URLs, e-mail address and @username replies
* Supported services: twitter, pownce, facebook, last.fm, del.icio.us, flickr
